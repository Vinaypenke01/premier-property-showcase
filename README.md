# Premier Property Showcase

A premium real estate portfolio website showcasing luxury properties with a focus on aesthetics and user experience.

## Project Info

This project is a modern web application built with React, TypeScript, and Tailwind CSS. It features a responsive design, smooth animations, and a curated selection of luxury real estate listings.

## Technologies Used

- **Frontend Framework**: [React](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/) & [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

## Getting Started

To run this project locally:

1.  **Clone the repository**
    ```bash
    git clone <YOUR_GIT_URL>
    cd premier-property-showcase
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

## Features

- **Luxury Property Listings**: Browse high-end properties with detailed descriptions and images.
- **Search & Filter**: Filter properties by location, type, and budget.
- **Responsive Design**: Optimized for all devices, from desktops to mobile phones.
- **Contact Form**: Integrated contact form for inquiries.

## deployment

You can deploy this application using any static site hosting service like Vercel, Netlify, or GitHub Pages.
